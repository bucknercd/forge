# Forge

Forge is a stateful design-to-build engine that transforms high-level system ideas into fully designed and implemented cloud systems. It operates in two layers:

1. **Design Layer**: Refines ideas into architecture, decisions, and milestones.
2. **Execution Layer**: Implements milestones by generating infrastructure, application code, and validation.

## Vision
Forge aims to provide an architecture-first approach to system design and implementation, ensuring auditability, reproducibility, and validation-driven development.

## Getting Started
- Explore the `docs/` directory for design artifacts.
- Use the `src/` directory for the Forge engine implementation.
- Refer to `examples/` for sample projects.

## Repository Structure
- `docs/`: Design artifacts (vision, requirements, architecture, decisions, milestones).
- `src/`: Python package for Forge.
- `tests/`: Placeholder for unit and integration tests.
- `examples/`: Example projects.
- `scripts/`: Utility scripts.
